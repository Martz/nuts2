# Fireworks Display Software Research

> Research document covering available fireworks show design and choreography software solutions, both proprietary and open source.
>
> Last updated: January 2025

## Executive Summary

The fireworks display software market is dominated by a few key professional players, with limited open-source alternatives. According to the American Pyrotechnics Association, the professional fireworks display industry has seen a **35% increase** in the use of digital choreography tools over the past decade. Modern software allows for precision timing down to fractions of a second and synchronization with music that was previously impossible to achieve consistently.

## Professional/Proprietary Solutions

### 1. Finale 3D

**Website:** [finale3d.com](https://finale3d.com/)

Finale 3D is marketed as "The World's Most Powerful Fireworks Design Software" and provides real-time 3D visualizations with automatic scripting features. It's designed to be "easy enough for backyard enthusiasts, powerful enough for world-class professionals."

#### Editions & Pricing

| Edition | Target User | Key Features |
|---------|-------------|--------------|
| **Lite** | Beginners, enthusiasts | Streamlined feature set, most cost-effective |
| **Hobbyist** | Semi-professionals | DMX support, video export, larger inventories |
| **Pro** | Professionals | Visual rack layouts, Google Maps, drone show visualization, custom reports |

- **Pricing Model:** Annual subscription only
- **Multi-license Discount:** 35% off additional Pro licenses, 15% off additional Hobbyist licenses

#### Key Features

- **50+ Supplier Catalogs** with thousands of 1.3G Display, 1.4G Consumer, and 1.4G Pro Line effects
- **Realistic HD simulations** with drag-and-drop interface
- **Automatic rack addressing** and firing system script generation
- **Export to 40+ firing systems** including DMX and flame units
- **Deep integration with Finale Inventory** for advanced tracking
- **Google Maps integration** (Pro edition)
- **Drone show visualization** (Pro edition)

#### System Requirements

- Windows 10/11 with 8GB RAM (16GB recommended)
- Dedicated GPU (NVIDIA/AMD) recommended
- Mac support via Parallels or Boot Camp

#### Firing Systems Supported

ACE Pyro, APEX, FireOne, Pyrodigital, ActionBase, Galaxis, and 35+ others. Also supports generic CSV and Excel formats.

### 2. FWsim

**Website:** [fwsim.com](https://www.fwsim.com/)

FWsim is a German-developed fireworks visualization and show design software with over 30,000 fireworks shows designed worldwide. It's available on Steam and as standalone software.

#### Editions & Pricing

| Edition | Price | Use Case |
|---------|-------|----------|
| **Basic** | Free/Low cost | Learning and simple designs |
| **Creative** | Mid-tier | Hobbyist choreography |
| **Pro** | €89/month or €799/year | Professional use |

#### Pro Features

**Show Design:**
- Online Fireworks Database (7,241+ effects)
- Google Maps integration for real 3D locations
- Advanced cue snapping and smart clone
- Import camera moves from other 3D software
- Chain creation for sequential effects

**Professional Tools:**
- Live price & NEM calculation
- Script view with firing systems export
- Automatic & manual cue addressing
- Multi-monitor support

**Effects & Output:**
- Flamejets and DMX support
- Drone show import
- Color variation tool
- Custom tail components
- Video export with effect previews

**Plans & Reports:**
- Flexible PDF plan generation
- Excel export
- Label printing

#### Licensing

- Usable on 2 computers (non-simultaneous)
- Files created in Pro cannot be opened in Basic/Creative editions

### 3. Visual Show Director (Infinity Visions)

**Website:** [infinityvisions.net](https://www.infinityvisions.net/)

Visual Show Director (VSD) was the first pyrotechnic choreography software, originally released in 1995. Now in version 8, it's considered the worldwide industry-leading fireworks simulation software for professional designers.

#### Key Capabilities

- Fireworks, special effects, lasers, light beams
- Image projections and object animation
- Multiple 3D sites with stereoscopic rendering
- Real-time and video render modes
- Direct DMX control via ShowDirector Control

#### Licensing Options

| Duration | Features |
|----------|----------|
| **Monthly** | Short-term projects |
| **1 Year** | Standard subscription |
| **3 Years** | Includes PyroCreator4D, DMX Control, SD Tools |

**Multi-computer Options:**
- 2 computers (not simultaneous): Save $390 on 36-month
- 2 computers (simultaneous): Save $600 on 36-month

#### Special Programs

- **Weekly/Monthly Rentals** for online use anywhere
- **Educational Licenses** for institutions
- **One-week Demo** fully functional trial

#### Professional Endorsements

Used in Olympic Game Ceremonies, live concerts, TV shows, and international pyromusical competitions for over a decade.

### 4. FireOne Software Suite

**Website:** [fireone.com](https://fireone.com/)

FireOne is the world leader in digital pyrotechnic firing systems, responsible for major displays including Sydney Harbour New Year's Eve, Burj Khalifa Dubai, and all Disney Theme Parks.

#### Software Components

| Software | Purpose |
|----------|---------|
| **ScriptMaker** | Choreography software for designing and timing shows to music |
| **CueMaker** | Assigns products to cues for firing system integration |
| **Timecode Wizard** | Creates digital SMPTE timecode from music scores |
| **UltraFire** | Field software for firing, monitoring, and real-time interaction |

#### Key Features

- Digital firing with exact timing precision
- Automatic cue assignment optimization
- Setup and testing routines before firing
- Real-time monitoring during shows

### 5. PyroDigital / Show Director

**Website:** [pyroinnovations.com](https://pyroinnovations.com/) | [infinityvisions.net](https://www.infinityvisions.net/pyrodigital-field-controller/)

PyroDigital was the first digital firing system and remains the most widely used worldwide, installed in more theme parks than any other system (Disney, Universal Studios, SeaWorld, Six Flags, Sentosa).

#### Show Director Software Features

- Handles complex shows with 10,000+ cue lines
- Fast and versatile scripting interface
- Multi-script management without confusion
- SMPTE timecode support (30 frames/second)
- Proprietary PD FSK timecode option

#### Theme Park Installations

- Disney Florida, California, Tokyo, Paris
- Universal Studios
- SeaWorld
- Six Flags
- Sentosa Singapore

## Free/Freemium Solutions

### 1. IGNITE Show Designer

**Website:** [designer.ignitefiringsystems.com](https://www.ignitefiringsystems.com/designer)

IGNITE Show Designer is a **completely free** online show design software developed by the team behind COBRA Firing Systems.

#### Features

- **Web-based** (Chrome required, works on PC/Mac)
- Design shows with up to 108 ignitions and 6 modules
- Optional music integration with waveform view
- Personal fireworks database with bulk import
- Manufacturer fireworks catalog access

#### Show Management

- List view and waveform view for event placement
- 4-digit show codes for sharing
- QR code generation for product packaging
- Show reports: module report, cue report, product list
- Avery 5160/5260 label printing

#### Integration

- Direct sync with IGNITE app
- Compatible with COBRA Show Creator
- Works with any SMPTE timecode control system

### 2. PyroCreator 4D

**Website:** [pyrocreator.com](https://www.pyrocreator.com/) | [infinityvisions.net](https://www.infinityvisions.net/software/pyro-creator-fx-generator/)

PyroCreator 4D is described as "the most intuitive pyrotechnic design and simulation software ever." It's a drag-and-drop adaptation of Visual Show Director for amateurs and as a sketch pad for VSD professionals.

#### Features

- Self-contained application
- Drag-and-drop interface
- Supports all effect types: flares, gerbs, strobes, comets, serpents, tourbillions, rockets, mines, roman candles, cakes, peonies, chrysanthemums, color change shells, multibreaks

#### Licensing Options

- Lifetime license
- Rent by week, month, or year

## Open Source Solutions

### 1. FireShow

**Repository:** [github.com/giuseppe-coco/FireShow](https://github.com/giuseppe-coco/FireShow)

FireShow is an open-source 3D fireworks display design and visualization software developed in C++ and OpenGL, inspired by professional tools like FWsim and Finale 3D.

#### Features

| Feature | Description |
|---------|-------------|
| **Advanced Particle System** | Simulates physics of various firework types including cascading effects (trails) |
| **Pre-defined Types** | Library of Peonies, Chrysanthemums, Willows, Volcanoes |
| **Firework Editor** | Dear ImGui interface for modifying color, velocity, particle count |
| **Interactive Timeline** | Place events with play, pause, reset controls |

#### Technology Stack

- C++
- OpenGL for 3D rendering
- Dear ImGui for UI

#### Status

Active open-source project with community interest (discussed on Hacker News, November 2025).

**Repository:** [sourceforge.net/projects/vfwpd](https://sourceforge.net/projects/vfwpd/)

A free pyrotechnics planning and designing visualization software available on SourceForge.

## Industry Trends

### Current Developments

1. **Drone Integration** - Software now supports coordinating fireworks with drone light shows
2. **Cloud-Based Design** - Web applications like IGNITE Designer enabling remote collaboration
3. **VR/AR Preview** - Emerging capability for immersive show previews
4. **AI-Assisted Choreography** - Experimental features for automatic beat detection and effect placement

### Market Growth

- 35% increase in digital choreography tool adoption over past decade
- Growing demand for precision timing in pyromusicals
- Increasing integration with entertainment control systems (DMX, SMPTE)
